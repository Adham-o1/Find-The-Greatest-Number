# Find The Greatest Number

برنامج بسيط بلغة C++ يقرأ ثلاث أعداد من المستخدم ويطبع الأكبر بينهم.

## حالة المشروع
مشروع تعليمي/تجريبي — مناسب لتعلم أساسيات الإدخال/الإخراج والتحكم الشرطي في C++.

## المتطلبات
- مترجم C++ (مثل `g++` من MinGW أو من WSL على ويندوز).  
- أو استخدم Visual Studio / VS Code مع إعدادات C++ المناسبة.

## الملفات المهمة
- `Find The Greadest Number.cpp` — كود المصدر.  
- `.gitignore` — يستبعد ملفات البناء وملفات VS Code من الرفع إلى GitHub.

## كيفية البناء والتشغيل (Windows باستخدام g++)
1. افتح Terminal داخل فولدر المشروع (في VS Code اضغط `Ctrl + \``).  
2. لتجميع الملف:
   ```bash
   g++ "Find The Greadest Number.cpp" -o FindTheGreatestNumber.exe
